# VitalCare Medical Clinic System

## Overview
VitalCare is a web-based medical clinic system that manages doctors, patients, admins, and medicines using IndexedDB.

The system allows users to interact with healthcare data through different roles such as Doctor, Patient, and Admin.

---

## Features
- Manage doctors, patients, admins, and medicines  
- Browser-based database using IndexedDB  
- Multiple user roles (Doctor, Patient, Admin)  
- Data storage and retrieval using web technologies  

---

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- IndexedDB  

---

## How to Run

### 1. Open Database Setup
- Open `VitalCareDB.html` in a browser (Google Chrome recommended)  
- You will see buttons like:
  - Create Database  
  - Get/Add Doctors  
  - Get/Add Patients  
  - Get/Add Admin  
  - Get/Add Medicines  

---

### 2. Create Database
- Click **Create Database**  
- This initializes:
  - Doctors  
  - Patients  
  - Admin  
  - Prescriptions  

- Verify using:
  - Right click → Inspect → Console  

---

### 3. Add Data
- Use the buttons to insert and retrieve data  
- View stored data in:
  - Inspect → Application → IndexedDB  

---

### 4. Run the Website
Open any of the following files:
- `index.html`  
- `admin.html`  
- `doctor.html`
